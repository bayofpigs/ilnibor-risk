Style Guide for this project:

-------------------------------------------------------------------------------------
Use camel-case for function/variable names:
ie. 

thisIsAFunction
thisIsAVariable
mySpecialVariable
-------------------------------------------------------------------------------------
Capitalize the first letter of each word for class names:
ie. 

ThisIsAClass
ClassesAreNice
Sample
-------------------------------------------------------------------------------------
Capitalize every letter for constant names and separate each word with an underscore:
ie.

MY_SPECIAL_CONSTANT
DONKEYS_GENERATE_ENERGY
FOR_THE_UNIVERSE
-------------------------------------------------------------------------------------
Favor putting a space between function parameters and choice and loop conditions:
ie. 

while (some condition)...
if (true/false)...
for (yeah = 0; yeah < ok; sureMan++)...
-------------------------------------------------------------------------------------
Put braces next to function parameters and choice and loop conditions:
ie. 

while (some condition) {
  // do stuff
  ...
}

if (some condition {
  // do stuff
  ...
}

public static void someFunction (arg1, arg2) {
  // do stuff
  ...
}
-------------------------------------------------------------------------------------
